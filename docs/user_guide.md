# StrathosNet User Guide

## Introduction
This user guide provides instructions on how to use the StrathosNet-Core application effectively.

## Getting Started
1. **Installation:** Follow the installation instructions in the README.md file.
2. **Configuration:** Configure your settings in the `config/` directory.

## Using the Application
### Logging In
- Navigate to the login page and enter your credentials.
- Upon successful login, you will be redirected to the dashboard.

### Managing Users
- To add a new user, navigate to the User Management section and fill out the required fields.
- You can edit or delete existing users from the same section.

### Monitoring Network Status
- Access the Network Status page to view real-time data on satellite connections and user activity.

## Conclusion
For further assistance, please refer to the troubleshooting section or contact support.
